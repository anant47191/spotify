# Spotify Clone

A simple, stylish music player inspired by Spotify. This project recreates the feel of a modern music app with a left library panel, playlist cards, a mini player bar, and smooth playback controls.

## What this project includes

- A clean Spotify-like layout
- Playlist-style album cards
- A music library panel on the left
- Play, pause, previous, and next controls
- Seek bar and volume slider
- Keyboard support for quick play/pause using the spacebar

## Tech Stack

- HTML
- CSS
- JavaScript

## Run locally

1. Clone the repository.
2. Open the project folder.
3. Start a local server, or simply open `index.html` in your browser.

For the best experience, use a local development server such as Live Server in VS Code.

## Project structure

- `index.html` — main layout
- `style.css` — overall styling
- `utility.css` — helper utilities
- `script.js` — music playback logic and UI interactions
- `songs/` — audio tracks used by the player

## Notes

This project is a front-end music player clone and is designed for local use. The songs in the `songs/` folder are loaded directly by the browser.
